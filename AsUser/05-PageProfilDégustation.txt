create_page_profil_degustation

## Description
En tant qu'utilisateur, je veux voir le profil de dégustation qui correspond à mes préférences.
En tant qu'utilisateur, je veux pouvoir revenir sur mes préférences si le profil ne correspond pas.
En tant qu'utilisateur, je veux pouvoir aller vers l'atelier de création.
En tant qu'utilisateur, je veux pouvoir aller vers la liste des vins qui correspond à mes préférences.

## Fonctionnalités
- Montrer un profil de dégustation
- Avoir une image qui redirigerait l'utilisateur/trice vers la page de l'atelier de création
- Avoir une image qui redirigerait l'utilisateur/trice vers la page listant les vins qui correspondrait à ses préférences.
- Avoir un bouton qui redirigérait vers la page de fiche de dégustation pour changer les préférences de l'utilisateur/trice

## Critères de validation
- Lorsque l'utilisateur/trice arrive sur cette page, le profil de dégustation doit correspondre à la somme de toutes les préférences enregistrées précédemment par iel.
- Quand l'utilisateur/trice clique sur l'image "Assemblez votre vin", iel sera redirigé(e) vers la page de l'atelier de création
- Quand l'utilisateur/trice clique sur l'image "Nous avons sélectionné pour vous", iel sera redirigé(e) vers la page listant les vins qui correspondrait à ses préférences.
- Quand l'utilisateur/trice clique sur le bouton "Revenir sur mes préférences", iel sera redirigé(e) vers la page de fiche de dégustation pour changer ses préférences.
